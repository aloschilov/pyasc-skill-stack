# add kernel (vadd) — asc2 API

pyasc kernel implementation for element-wise vector addition: out = x + y.

## Files

- `kernel.py` — Kernel implementation using asc2 tile-based API
- `docs/design.md` — Design document
- `docs/environment.json` — Environment snapshot (generated by verify_environment.sh)
- `test/` — Test data directory

## Usage

```bash
# Set up CANN env first — see docs/cann-setup.md
source $HOME/Ascend/cann/set_env.sh
export LD_LIBRARY_PATH=$ASCEND_HOME_PATH/tools/simulator/Ascend950PR_9599/lib:$LD_LIBRARY_PATH

python3.10 kernel.py -r Model -v Ascend950PR_9599   # Run with simulator
python3.10 kernel.py -r NPU                         # Run with NPU hardware
```

Or via pytest:

```bash
pytest kernel.py --backend Model --platform Ascend950PR_9599
```

## Verification

The kernel verifies output with `np.testing.assert_allclose(out, x + y)`.
